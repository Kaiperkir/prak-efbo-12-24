# Контрольная работа №2: Fullstack-приложение "Интернет-магазин"

**Группа:** ЭФБО-12-24  

## Описание проекта
Данный проект представляет собой полноценное веб-приложение интернет-магазина, состоящее из backend-части на Node.js/Express и frontend-части на React. В рамках контрольной работы были объединены и доработаны материалы практических работ (со 2-й по 11-ю).

Приложение реализует ролевую модель доступа (RBAC), JWT-авторизацию, работу с файловой системой (загрузка изображений) и хранение данных в формате JSON.
